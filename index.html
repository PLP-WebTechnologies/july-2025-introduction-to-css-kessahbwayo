About CSS

CSS is used to style HTML elements by changing text color, size, layout, and spacing. It’s also how we implement the Box Model for padding, margin, and borders.

The Box Model

Every HTML element is a box, made of:

  • Content – the text or image inside
  • Padding – space inside the border
  • Border – the edge wrapping the element
  • Margin – space outside the border
